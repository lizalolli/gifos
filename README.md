# Gifos
Gifos es un buscador de gifs que muestra gifs desde la API de Giphy.

## ¿Qué puedes hacer en la aplicación?
Desde Gifos puedes:
- buscar gifs con el buscador
- seleccionar gifs sugeridos
- cambiar el tema de la aplicación a 'modo dark'
